# Detector de Gas con Arduino y Sensor MQ-7

Proyecto desarrollado para la materia de Sistemas Embebidos.

## Descripción

Este proyecto implementa un sistema de monitoreo de gases utilizando un sensor MQ-7 conectado a un Arduino UNO.

El sistema muestra la lectura del sensor en una pantalla LCD 16x2 y genera diferentes niveles de alerta mediante LEDs y un buzzer dependiendo de la concentración detectada.

## Características

- Lectura analógica del sensor MQ-7.
- Visualización de datos en LCD 16x2.
- Indicador visual mediante LEDs.
- Alarma sonora mediante buzzer.
- Sistema de tres niveles de alerta.
